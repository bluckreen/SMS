📌 Project Overview

The Student Management System (SMS) is a web-based application designed to efficiently manage student data within an institution.

It provides a centralized platform to handle student registration, attendance, academic records, and documents, making data management simple and organized.

❗ Problem Statement

Managing student data manually can be:

Time-consuming
Prone to errors
Difficult to organize and access
Inefficient for tracking academic progress

💡 Solution

The system solves these problems by offering:

A centralized digital platform for all student data
Easy data entry and retrieval
Automated attendance and academic tracking
Organized student profiles and records


🛠️ Technologies Used

Backend: Python (Flask)
Database: SQLite3
Frontend: HTML, CSS, Bootstrap
Scripting: JavaScript
Tools: VS Code, Git


🚀 Key Features

📝 Student Registration
Add and manage complete student details
📊 Attendance Tracking
Track attendance with percentage calculation
📋 Marks & Grades
Store academic performance data
📁 Document Uploads
Manage student documents securely
👤 Student Profiles
View all student information in one place
📚 Library System
Track book issue and return
